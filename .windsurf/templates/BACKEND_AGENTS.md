# Backend Instructions

> Applies to all files in this directory and subdirectories.

## Tech Stack
- **Framework:** NestJS
- **Language:** TypeScript
- **Database:** PostgreSQL (TypeORM/Prisma)

## Conventions
- **Controllers:** Thin, only HTTP handling.
- **Services:** Business logic here.
- **DTOs:** Must use `class-validator`.
- **Naming:** `user.service.ts`, `auth.controller.ts`.
